# Social Media Aplication

This project is build on node.js and express.js
There server methods which i have implemented like, Authorization, Authunatication, etc
This project has defferent User intractions like addding a post, adding commemt to the post, liking that post 
etc.

To run this project in local, please follow these instructions
1. clone the project from https://github.com/joewish/basicBackEndSocialMediaAPP.git
2. execute the following command "node server.js"
3. Then the project will be runing locally and you will be able to see the log "server is listening at port 3000".
